# Overview

Demonstration of the integration of git, GitHub, Python, and Visual Studio code. A simple "Hello World" is printed when run.

Software demo video can be found [here](https://youtu.be/3dxseuF1UAI).

# Development Environment

* Visual Studio Code
* Python 3.9.0 64-bit
* Git/GitHub
